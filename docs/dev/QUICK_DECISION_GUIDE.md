**推荐: 选项 C - 混合方案**

```sql

    type VARCHAR(50),              -- password/server/database/service
    host VARCHAR(255),              -- IP 或域名

    environment VARCHAR(50),        -- dev/test/prod
    metadata JSON,                  -- 类型特定字段
```
